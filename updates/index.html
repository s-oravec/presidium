v0.1.6

¶

Summary

6 June 2017 General improvements and features that include support for semantic versioning, Javadoc ingestion and link validation.

New Features

  • Semantic Versioning: Add support to publish and switch between different site versions
  • Javadoc Import: Java doclet to import source code comments to Presidium
  • Link validation: Tool to check for any broken links
